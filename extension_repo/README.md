# Extension Repository Artifacts

This directory stores Blender Extensions distribution artifacts:

- `*.zip` packages generated by `blender --command extension build`
- `index.json` generated by `blender --command extension server-generate`

Use:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/release_new_version.ps1 -Version X.Y.Z
```

The script will populate this directory automatically.
